Matrix selector

With the `matrix selector` widget the underlying matrix of the heatmap can be selected which is based on a sector*indicator result. The selected matrix is based on the following configuration attributes:

  • `perspective=direct` => matrix D
  • `perspective=final` => matrix U
  • `perspective=direct&analysis=consumption|production` => direct result
  • `perspective=final&analysis=consumption|production` => final result

See the code snippets of the widget for what these matrices mean and how they can be calculated. Note that the matrices D and U of a multi-regional model do not really make sense in the heatmap as the same sectors in different regions are aggregated to a single sector in the heatmap in this case. 

            
                var config = useeio.urlConfig();
                var selector = useeio.matrixSelector({
                    selector: '#matrix-selector',
                });
                config.join(selector);
