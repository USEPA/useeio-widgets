USEEIO Widgets

A collection of embeddable components for visualizing data from the USEEIO API.

Local JSON files may be generated from the API or you can use pre-generated files.

Sector List Widget
Commodity data from the U.S. Bureau of Economic Analysis (BEA)
  • Sector List Alone
  • Sector List and Mosaic Heatmap
  • Sector List and Mosaic Heatmap - with positive indicator values
  • Sector List and Mosaic Heatmap - with all values
  • Sector List with Inflow and Outflow - Rubber tire manufacturing example
  • Sector List with Inflow and Outflow - Rubber tire with settings filter
Impact Chart Widgets
Environmentally-Enabled Input-Output Comparison
  • Impact Bar Chart - with sector list and settings filter and stand-alone
  • Environmental Profile Chart - Vertical Bar Chart
Filters and Configuration Test
Communication occurs between widgets using hash values. Test your setup.
  • Model Configuration Test
  • Settings Filter
  • Settings Display Widget
